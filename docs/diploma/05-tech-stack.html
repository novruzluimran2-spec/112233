Таблица / рисунок — Технологический стек и архитектура веб-приложения «Antalya»

Браузер
HTML + CSS + JS
Laravel 13 (PHP 8.3)
Blade + REST API
MySQL
Eloquent ORM
Railway
FrankenPHP / Railpack

Backend

  • PHP 8.3, Laravel Framework 13
  • Eloquent ORM, миграции, сидеры
  • REST API: /api/menu, /api/orders, /api/reservations
  • Админ-панель: Laravel Auth, bcrypt
  • Расширение pdo_mysql

Frontend (клиентская часть)

  • Blade-шаблоны → статические HTML-страницы
  • public/styles.css — оформление
  • public/app.js — меню, корзина, тема
  • Корзина: localStorage
  • Tailwind 4 + Vite (сборка в dev)

База данных

  • MySQL (production, Railway)
  • SQLite (локальная разработка)
  • Таблицы: menu_*, orders, order_items, reservations, users

Развёртывание

  • Railway (Railpack, FrankenPHP)
  • start-container.sh: migrate + db:seed
  • GitHub: CI (PHPUnit)
  • Переменные: APP_KEY, DB_*, MYSQL_*
